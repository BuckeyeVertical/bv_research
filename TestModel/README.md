# setup

### pull / clone repo
```bash
git clone https://github.com/BuckeyeVertical/bv_research.git
or git pull

cd bv_research
cd TestModel
```
### setup python venv
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
